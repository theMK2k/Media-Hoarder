# IMDB Scraper

These are dumps fetched by imdb-scraper-watchdog by running:

```
npm run imdb-scraper-watchdog -- --dumpScrapedHTML
```

## V1 (2020-12-12)

The V1 imdb dumps were creted on 2020-12-12. The dump filenames do not contain a `-v1` suffix.
